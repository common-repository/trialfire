Inserts Trialfire tracking snippet in all pages.

== Description ==

This plugin adds the required tracking snippet for Trialfire in all pages.

For more information visit:

[Trialfire](http://www.trialfire.com)

== Installation ==

1. Upload `trialfire` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy your tracking snippet from app.trialfire.com and paste it in the text field (Admin > Settings > Trialfire)
